# Font Installation Instructions for Custom Header

## Required Font Files

1. **Cinzel SemiBold (600)**
   - Download from: https://fonts.google.com/specimen/Cinzel
   - Click "Download family"
   - Extract ZIP and find: `Cinzel-SemiBold.ttf` or `Cinzel-600.ttf`
   - Copy to: `assets/fonts/Cinzel-SemiBold.ttf`

2. **Montserrat SemiBold (600)**
   - Download from: https://fonts.google.com/specimen/Montserrat  
   - Click "Download family"
   - Extract ZIP and find: `Montserrat-SemiBold.ttf` or `Montserrat-600.ttf`
   - Copy to: `assets/fonts/Montserrat-SemiBold.ttf`

## Next Steps After Fonts Are Added

Once both .ttf files are in `assets/fonts/`:
1. Fix corrupted dashboard.tsx
2. Update _layout.tsx to load fonts
3. Add centered header text: "WANDERLUST" / "TRAVEL COMPANION"
